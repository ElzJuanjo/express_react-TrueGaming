## DEPENDENCIAS

###### Backend:

* express
* mysql
* nodemon
* dotenv
* cors

###### Frontend:

* react-router-dom
* --save-dev @babel/plugin-proposal-private-property-in-object

## COMANDOS

###### Instalar dependencias: ``npm i <module>``

###### Iniciar servidor: ``nodejs index.js``

###### Iniciar web: ``npm start``
