## DEPENDENCIAS

###### Backend:

* express
* pg
* nodemon
* dotenv
* cors
* bcrypt
* nodemailer
* crypto

###### Frontend:

* react-router-dom
* --save-dev @babel/plugin-proposal-private-property-in-object

## COMANDOS

###### Instalar dependencias: ``npm i <module>``

###### Iniciar servidor: ``nodejs index.js | npm run start``

###### Iniciar web: ``npm start``
